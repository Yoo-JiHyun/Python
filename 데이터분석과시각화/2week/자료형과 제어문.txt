자료형
    자료형 종류
        - 숫자형 : 0 ~ 9와 소수점으로 구성되는 데이터
        - 문자형(문자열) : 따옴표로 묶여진 데이터(순서가 있고 정수로 인덱싱하며 길이가 있음)
        - 리스트(list), 튜플(tuple) : 여러 가지 숫자형과 문자형 자료로 구성된 데이터(순서가 있고 정수로 인덱싱하며 길이가 있음)
        - 딕셔너리(dict) : Key, Value로 구성되는 단순한 구조의 데이터
        - 논리 : 참(True), 거짓(False) 두 가지로 구성되는 데이터
        - 세트(set) : 집합의 특징을 갖는 데이터 (중복 비허용)

제어문
    제어문 종류
        반복 : for, while
        비교 : if
            - elif, else 등과 함께 사용하여 여러 가지 경우 비교 가능
            - break : 반복 중단
            - continue : 반복 계속
